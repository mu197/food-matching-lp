食品寄附マッチングプラットフォーム｜第2弾実証実験

食品ロスとして消えてしまうはずだった食品を、タイムリーに、スピード感をもって食料支援団体へ。 大規模フードバンクから、フードパントリー、こども食堂まで—— 賞味期限切迫品も無駄にしない、新しい食品寄附の仕組みに参加しませんか。

PROJECT

① プロジェクトについて

日本では、年間約464万トンの食品が廃棄される一方で、必要な食品が届かない人々がいます。 この矛盾を、テクノロジーとネットワークの力で埋めるのが、私たちのミッションです。

第2弾実証実験──
"食品ロス × 食料支援"の最後のピースを埋める。

食品寄附マッチングプラットフォームは、食品提供企業と食料支援団体を オンラインで瞬時につなぐ、日本初の仕組みです。第1弾実証実験では、大規模フードバンクとの 物流連携モデルを検証してきました。

第2弾では、対象を一気に拡大します。大規模フードバンクから、地域のフードパントリー、 こども食堂まで——これまで届けられなかった「賞味期限切迫品」や「少量品」も、 配達・引き取り可能な距離感の支援団体へ、迅速にマッチングします。

食品ロスを"出さない"のではなく、"無駄にしない"。 そのための社会インフラを、皆さまと一緒に育てていきたい。 ぜひ、この実験への参加をご検討ください。

VISION

② 私たちがめざす世界

"出てしまった"食品ロスを、"必要としている人"のもとへ。
タイムリーに、スピード感をもって。

食品企業からイレギュラーで出てしまった食品ロス、
防災備蓄品の賞味期限切迫品、農家さんの規格外野菜──
その情報をリアルタイムで集約し、食料支援団体にマッチング。
より多くの食品ロスを、より多くの必要としている人へ。

⏱️

TIMELY
タイムリーに

賞味期限が迫る食品も、リアルタイムマッチングで"間に合わせる"。情報公開から最短当日マッチング。

SPEEDY

大規模フードバンクから、地域のフードパントリー、こども食堂まで。配達・引き取り可能な距離感の寄付を実現。

HOW TO USE

④ システムの使い方

登録から運用まで、特別な機材は一切不要。
スマホ・PCのブラウザがあれば、どなたでもすぐにご利用いただけます。

FOR DONOR

寄付者（提供企業・自治体・農家）

  1. 無料アカウント登録企業情報・拠点情報を入力（最短10分）
  2. 食品情報を投稿品目・数量・賞味期限・写真をアップロード。テンプレ化で繰返し投稿も簡単
  3. マッチング結果を確認受取団体からのリクエストを確認・承認
  4. 引渡し方法を選択①配送依頼／②受取団体による引取／③物流事業者連携 から選択
  5. 引渡し完了 → 寄付実績ダウンロードCSR報告書・税控除書類に活用可能なPDFを発行
FOR RECIPIENT

食料支援団体（フードバンク・パントリー・こども食堂）

  1. 無料アカウント登録団体情報・受取可能エリア・受入条件を登録
  2. 食品情報を閲覧近隣で出た食品をリアルタイムにマップ＆リスト表示
  3. 受取リクエスト必要数量・引取可能日時を送信（複数団体間で先着 or 配分）
  4. 受取・配送調整寄付者と直接または事務局経由でやりとり。配送依頼も可能
  5. 受領報告 → 配布受領数量を記録し、利用者へ配布。実績はダッシュボードで一元管理
